## Why

The XAdmin documentation site is missing a comprehensive configuration guide. While the sidebar references `/guide/configuration` in the navigation, this page does not exist. Users need a central reference for all available configuration options for the XAdmin Laravel package.

## What Changes

- Create `/docs/guide/configuration.md` with comprehensive configuration documentation
- Create `/docs/zh/guide/configuration.md` with Chinese translation
- Document all configuration options available in the XAdmin package
- Include code examples for common configuration scenarios
- Mark the existing sidebar link as functional (it currently points to a non-existent page)

## Capabilities

### New Capabilities
- `config-docs`: Complete configuration reference documentation for XAdmin package

### Modified Capabilities
- None (no existing capability requirements are changing)

## Impact

- New documentation files in `docs/guide/` and `docs/zh/guide/`
- No code changes required
- No breaking changes
- Improves user onboarding and discoverability of configuration options
